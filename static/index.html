About This Project

This application serves as a proof-of-concept for deploying Automatic Speech Recognition (ASR) models for low-resource languages, specifically Akan and Sindhi. The models were fine-tuned on the Whisper architecture.

Architecture: A custom HTML/JS frontend styled with a warm, modern palette, communicating with a FastAPI backend. The backend utilizes the transformers library to perform local, on-server model inference without relying on external APIs.
